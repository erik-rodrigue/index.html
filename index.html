O QUE É PROGRAMAÇÃO?

É uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas.

O QUE É HTML?

Linguagem de Marcação de Hipertexto (HyperText Markup Language), é o componente básico da web. Ele é usado para inserir conteúdo e estabelecer a estrutura básica de um site. As tags do HTML definem como o conteúdo, como texto, imagens e vídeos, é exibido em um navegador.
